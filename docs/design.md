# 設計書 v1 — JISOO AI 月次数字ダッシュボード（AI経営参謀つき）

最終更新: 2026-06-13／状態: 仁義さんの最終OK待ち

## 目的

- 月次の経営数字を1画面で見える化し、月1の加藤さん（M&A）面談の定例資料にする
- 7月第1週に6月分を入力し、2026-07-08 13時・天神の対面面談で見せる（締切）
- 「AIの会社らしい」体験＝数字を入れるとAIが裏で経営分析を済ませている

## 利用者と権限

| 利用者 | 権限 |
|---|---|
| 仁義さん・経営メンバー | ログインして入力＋閲覧 |
| 加藤さん（社外・M&Aアドバイザー） | 閲覧専用ログイン。入力フォームは見えない |

認証は Supabase Auth（メール＋パスワード）。URLを知っているだけでは何も見えない。

## 画面（3つだけ）

1. **ログイン** — メール＋パスワード
2. **ダッシュボード** — 上から順に:
   - 今月サマリーカード（請求額・利益・年間契約社数・プロダクト完成数）
   - AI経営参謀ゾーン（下記3枚のAIカード）
   - 月次推移グラフ（利益の折れ線）
   - 月次一覧表（6月〜12月、列が増える）
3. **入力フォーム** — 月を選んで数字を入れて保存ボタン1回。スマホ対応（inputmode="numeric"）。未回収・利益は自動計算なので入力欄に出さない

## AI経営参謀（v1の3機能）

すべて「保存ボタンの後に裏で自動実行」。**利用者の追加入力はゼロ**。1回のEdge Function呼び出しでまとめて生成しコストを抑える。

| カード | 中身 | 失敗時 |
|---|---|---|
| 月次コメント | 前月比の変化と注目点を経営者向けに3行 | 「準備中です」表示＋再生成ボタン |
| M&Aウォッチ | 加藤さんの3条件（ストック売上・属人性・希少性）の観点で強み1つ＋改善点1つを100字 | カードごと非表示 |
| 目標到達予測 | 年内10社・30個への到達ペースをJS計算（線形回帰）＋AIの一言。**計算部分はAIなしでも出る** | AIの一言だけ非表示・数値バッジは残る |

データの流れ: フォーム保存 → Supabase書き込み完了 → Edge Function（今月＋過去データを集めてAnthropic API呼び出し）→ `ai_comments` テーブルに保存 → ダッシュボードが表示。

**鉄則: AIが全部落ちてもダッシュボードの数字表示は無傷（AIはおまけ層）。**

## やらないこと（過剰設計の防止）

- チャットボット常駐 — 「画面を見れば分かる」を壊す。コストも跳ねる
- 自動メール・Slack通知 — 誤送信リスク。共有は「URLを開いてもらう」で足りる
- 数字の入力提案・自動補完 — 請求額・入金額はAIに推測できない。フォームはシンプルなまま守る（Notion財務DBの失敗を繰り返さない）

## データの持ち方（Supabase）

```
monthly_metrics: year_month(一意キー "2026-06") / billing / payment / expense /
  companies_trained / annual_contracts / instructors / ai_products /
  prospects / confirmed_revenue / memo / created_at / updated_at
ai_comments: year_month / monthly_comment / ma_watch / forecast_note / generated_at
```

未回収（billing−payment）と利益（payment−expense）は保存せず表示時に計算（不整合防止）。

## インフラと分離方針（重要）

- **新規GitHubリポジトリ＋新規Supabaseプロジェクト**。アートメイク事業（line-clinic-system／顧客個人情報あり）とは完全分離
- 公開は GitHub Pages。実証済みのCI自動デプロイのパターンを流用
- Anthropic APIキーは Supabase の secrets に保存（HTMLに書かない）
- 費用: Supabase無料枠＋GitHub無料枠＋AI APIが月数百円程度

## 実装の進め方

- コード実装はSonnetのサブエージェント中心（トークン節約）、設計判断・レビューは上位モデル
- 実装後は動作確認（ログイン→入力→保存→AIカード表示→閲覧専用アカウントで見え方確認）まで済ませてからURLを渡す
